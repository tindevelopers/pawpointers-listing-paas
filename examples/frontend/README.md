# Frontend Examples

This directory contains examples showing how to use the listing platform SDKs in your Next.js application.

## Examples

### customization-levels.tsx

Demonstrates all 4 customization levels:
1. **Level 1**: Use components as-is (quick start)
2. **Level 2**: Customize with props
3. **Level 3**: Use headless components + custom styling
4. **Level 4**: Use hooks only + build everything

### theme-override.tsx

Shows different methods for overriding design tokens:
- CSS variable overrides
- Tailwind config overrides
- Component-level overrides
- Theme wrapper pattern

## Usage

Copy these examples into your Next.js app and customize as needed.

## Next Steps

1. Review the examples
2. Choose your customization level
3. Copy relevant code to your app
